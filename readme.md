·现行Reg NMS与Reg ATS是否对加密交易构成不成比例的合规成本 
·是否需要为“加密ATS”设立专门的Form ATS或调整现有披露制度 
·加密ATS的信息披露是否应继续非公开，还是引入SEC审核或公开机制 
·在区块链和链上数据可追溯背景下，是否有必要保留Form ATS-R季度报送要求 
·是否需要明确非美元资产换算美元的合规方法 
·如何处理交易信息保密、系统风控（Rule 15c3-5）及系统合规（Reg SCI）等要求 
·如何在监管中避免阻碍个人开发软件、使用自动化或去中心化方式进行交易
